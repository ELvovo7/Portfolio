#### Описание:
Проверка сайта строительной компании

#### Проводимые проверки:
<li>Корректность отображения на устройствах</li>
<li>Работа ссылок и кнопок</li>
<li>Корректность отображения контента, включая изображения, таблицы и графики</li>
